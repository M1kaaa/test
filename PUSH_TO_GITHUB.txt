Как отправить проект на GitHub
================================

Способ 1 — скрипт с токеном (рекомендуется)
-------------------------------------------
1. Создайте токен: https://github.com/settings/tokens
   - "Generate new token (classic)", scope: repo.
2. Запустите:   запустить_гитхаб.bat
   Или:         powershell -ExecutionPolicy Bypass -File upload_with_token.ps1
3. Вставьте токен по запросу. Скрипт создаст репо patch-cord-calculator и запушит код.

Способ 2 — вручную
------------------
1. Создайте репозиторий на https://github.com/new
   Имя: patch-cord-calculator. Без README, без .gitignore.
2. В папке проекта:
   git remote add origin https://github.com/VASH_LOGIN/patch-cord-calculator.git
   git push -u origin main
3. Пароль = Personal Access Token (не пароль от аккаунта).
